# USTC-Register
A simple flask app for event registration.

*** WARNING: THIS IS A DEMO VERSION UNDER DEVELOPING. ***

# Init

modify the `SECRET_KEY` in `config.py`

```
python db_create.py
python db_migrate.py
python db_upgrade.py
```

# Run
```
python ustc_register.py
```
